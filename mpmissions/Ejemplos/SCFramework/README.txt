BDC's Server-Client Framework	Written for Arma 3	Jan 2018
Designed to only be run on dedicated server setups!
Installation instructions:
- Place the SCFramework folder from the archive into the mission folder's root directory
- Modify your Init.SQF file and add this line, preferrably towards the top:
  [] execVM "SCFramework\Init_SCFramework.SQF";
- Refer to the configuration options within the Config_SCFramework file to custom tailor this package to your server's needs
